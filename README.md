# TDD_UI_Framework
Basic TDD testing framework for UI with Java 8
created by Salih Aydin

Used External Libraries:
-Selenium
-Apache POI
-WebDriverManager

Plugins:
-Surefire Plugin (Reports and parallel execution)

Design Models:
-Page Object Design Model
-Singleton Design Pattern

Utility Classes
-Config Reader
-Data Provider
-Dropdown Utils
-Excel Utils
-Table Utils
-Wait Utils
-Window Handle Utils
