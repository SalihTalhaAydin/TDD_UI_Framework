# TDD_UI_Framework
Basic TDD testing framework for UI with Java 8

Used External Libraries:
-Selenium
-Apache POI
-WebDriverManager

Plugins:
-Surefire Plugin (Reports and parallel execution)

Design Models:
-Page Object Desing Model
-Singleton Desing Pattern

Utility Classes
-Config Reader
-Data Provider
-Drop Down Utils
-Excel Utils
-Table Utils
-Wait Utils
-Window Handle Utils
